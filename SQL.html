SQL

Wat is SQL?

SQL staat voor Structured Query Language, wat in het Nederlands vrij vertaald kan worden naar “Op een gestructureerde manier de database aanspreken”. Het is een programmeertaal die is ontworpen om met databases te communiceren. SQL is vooral achter de schermen aanwezig en wordt gebruikt in en rondom websites voor onder andere het opslaan van gegevens in een database, maar ook het opvragen van gegevens uit deze database.
